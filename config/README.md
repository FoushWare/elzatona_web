# Configuration Files

This directory contains deployment and service-specific configuration files:

## Firebase
- `firebase.json` - Firebase project configuration
- `firestore.indexes.json` - Firestore database indexes
- `firestore.rules` - Firestore security rules

## Deployment
- `vercel.json` / `vercel-main.json` / `vercel-storybook.json` - Vercel deployment configurations

## Components
- `components.json` - shadcn/ui components configuration

## Note
Core development configuration files (Next.js, ESLint, Jest, TypeScript, Tailwind, PostCSS, Vitest) remain in the root directory as they are required by the build tools.
