# Quickstart: 008 Security Fixes Plan

## 1) Review task list

- Open `specs/008-security-fixes-plan/tasks.md`
- Execute in phase order; respect task dependencies.

## 2) Apply code/config updates

- Keep changes scoped to each task phase.
- Run targeted checks where practical before moving on.

## 3) Track completion

- Mark finished tasks as `[X]` in `tasks.md`.
- Keep remaining items unchecked until fully verified.

## 4) Final validation

- Run build/lint/security checks listed in final phase.
- Summarize pass/fail and unresolved items.
