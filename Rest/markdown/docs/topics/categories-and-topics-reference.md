## Question Types

The system supports the following question types:

- `multiple-choice` - Single or multiple correct answers
- `open-ended` - Free-form text responses
- `true-false` - Boolean questions
- `code` - Code-based questions with test cases

## Difficulty Levels

Questions can be categorized by difficulty:

- `beginner` - Basic concepts, entry-level
- `intermediate` - Moderate complexity, some experience required
- `advanced` - Complex topics, expert-level knowledge

## Usage Notes

- **Categories** are the main classification (e.g., "JavaScript Core", "React")
- **Topics** are specific subjects within categories (e.g., "Closures", "useState")
- **Learning Paths** are structured learning sequences
- All fields are optional - questions will work with minimal metadata
- Use exact names as shown above for consistency
- Tags can be added for additional categorization (e.g., "#javascript", "#interview")
